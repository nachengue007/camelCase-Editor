# Editor de Textos

## Descripción

Editor de textos simple y eficiente desarrollado en Rust. Proporciona una experiencia ligera para edición de archivos con una interfaz intuitiva.

## Características

- Creación y edición de archivos de texto
- Interfaz de usuario clara y responsiva
- Soporte para múltiples codificaciones
- Guardado automático de cambios
- Atajos de teclado personalizables

## Requisitos

- Rust 1.70 o superior
- Cargo

## Instalación

```bash
cargo build --release
```

## Uso

```bash
cargo run
```
